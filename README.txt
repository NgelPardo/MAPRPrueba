Evaluacion tecnica de desempeño

La siguiente aplicacion tiene como funcion llevar el control de clientes de dos empresas que
buscan unir sus recursos. Con el fin de enviar promociones a sus clientes frecuentes la 
aplicacion gestionara el estado de los clientes, la aplicacion cuenta con funcionalidades
como la de busqueda por ID o la actualizacion de las fechas de las ultimas compras. Teniendo un registro de las compras realizadas se decidira que clientes son frecuentes. Ademas se diseño la base de datos para cada una de las empresas.
